# 💀 TERMIDEATOR

> **Tu idea entra. Sale dinero. Sin preguntas.**

El agente más brutal para Claude Code. Convierte cualquier idea en un **blueprint ejecutable de 20 secciones** con validación de mercado, stack recomendado, seguridad integrada, métricas, riesgos identificados y plan de launch de 90 días — antes de escribir una línea de código.

De idea vaga a SaaS listo para lanzar en 1 conversación.

---

## ⚡ Quick Start

```bash
git clone https://github.com/[tu-usuario]/termideator.git
cd termideator
# Abre Claude Code en esta carpeta → lee el CLAUDE.md automáticamente
# Luego: "Quiero construir un SaaS de invoicing. Ayúdame a diseñarlo."
```

---

## 🎯 El Problema que Resuelve

La mayoría de agentes te generan código directo. **TERMIDEATOR diseña primero, mata después.**

```
Tu idea vaga
    ↓
[Phase 1] 2-3 preguntas que importan
    ↓
[Phase 2] 4 sub-agentes paralelos (mercado, dinero, tech, riesgos)
    ↓
[Phase 3] Confirmación arquitectura
    ↓
[Phase 4] Blueprint 20 secciones LISTO para Claude Code construir
```

El blueprint resultante es tan completo que **Claude Code puede construir TODO sin preguntarte nada.**

Todos corren simultáneamente. TERMIDEATOR integra los reportes en la arquitectura final.

## 🗺️ Blueprint de 20 Secciones

1. Executive Summary
2. **Business Model** — PROFIT_MATRIX, unit economics (CAC/LTV), KILLER_MOVE_24H
3. Target Users & Jobs-to-be-done
4. **Competitive Advantage** — TAM/SAM/SOM + moat
5. **Data Model** — Esquema + RLS policies SQL-ready
6. API Design
7. **Design System** — 55 referencias reales (Stripe, Notion, Apple, Linear...)
8. File & Storage Strategy
9. **Tech Stack** — Stack recomendado con rationale
10. Environment Setup — CORS + security headers
11. Authentication & Authorization
12. Payment Integration
13. Testing Strategy
14. **Error Handling + Risk Matrix** — 20 riesgos + failure modes
15. Skills to Use During Build
16. **Non-Negotiable Rules** — 15 reglas de seguridad y calidad
17. Build Order — 15-25 pasos con time estimates
18. **Analytics & Metrics** — AARRR + PostHog events + weekly ritual
19. Growth Hooks — Viral loops, SEO, community
20. **Post-Launch 90-Day Roadmap** — Week 1 → Month 3

## 🚀 Casos de Uso

**Para Fundadores:**
- Tienes idea, no sabes por dónde empezar
- Quieres blueprint ejecutable antes de gastar dinero en developers
- Necesitas validar si el mercado existe sin 2 meses de research

**Para Developers Freelance:**
- Recibiste un blueprint de TERMIDEATOR
- Copias el CLAUDE.md del target project
- Construyes sin ambigüedad, sin preguntas de clarificación

**Para Escuelas / Bootcamps:**
- Enseña a estudiantes a diseñar, no solo a codear
- TERMIDEATOR es el proyecto final: genera blueprint, luego código
